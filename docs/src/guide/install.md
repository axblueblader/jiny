# Installation

## Environment

- JDK 1.8 or higher
- Gradle/Maven build configuration

## Gradle Project

`build.gradle`

```groovy
dependencies {
    compile group: 'com.jinyframework', name: 'jiny', version: '0.2.2'
}
```

## Maven Project

`pom.xml`

```xml
<dependency>
  <groupId>com.jinyframework</groupId>
  <artifactId>jiny</artifactId>
  <version>0.2.2</version>
</dependency>
```
